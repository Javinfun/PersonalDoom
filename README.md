# PersonalDoom

A simple Doom-inspired game engine built with Python and Pygame. This project demonstrates the foundations of a custom game loop, player movement, map rendering, and real-time game updates.

## Features

- Game loop with real-time updates
- Player movement and controls
- Basic map rendering
- FPS display in the window title
- Event handling and graceful exit system
- Modular project structure

## Project Structure

- `main.py` — Main game loop and initialization
- `settings.py` — Game configuration and constants
- `map.py` — Map rendering and world data
- `player.py` — Player movement and logic
